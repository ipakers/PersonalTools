# Personal Tools
## Start

0. Download Commandline tools:
run:

xcode-select --install

2. Clone the Repo

git clone https://github.com/ipakers/PersonalTools

3. run:

./configure.sh

4. You may need to change the brew location. By default it is...

/opt/homebrew/brew

but you may need to change it to...

/usr/local/bin/brew
